# WebAsset-Optimizer
Asset optimizer for web. Compress meshes and textures for fast loading times.

You can try it out in https://webasset.alternaterealitystudio.com/

For web I recommend to use 512px or 1024px textures.

ThreeJS can load Draco compressed meshes and Meshop, just make sure to read the documentation. 
Draco : https://threejs.org/examples/jsm/libs/draco/

Texture compression to webp is native but KTX2 requires it's own loaders:
KTX2: https://threejs.org/docs/#KTX2Loader
